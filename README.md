# Jackie Mohanty Portfolio

A modern, responsive, and professional portfolio website showcasing Jackie Mohanty's expertise as a Technology Consultant, Product Strategist, and Founder.

## Overview

This portfolio is a comprehensive digital showcase built with React and TypeScript. It reflects Jackie's professional journey, technical skills, projects, and vision as a technology leader. The site follows modern web development best practices and emphasizes clean design, performance, and user experience.

## Key Features

- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Modern Tech Stack**: Built with React 18, TypeScript, and Tailwind CSS
- **Animation & Transitions**: Smooth scrolling and micro-interactions
- **Project Showcase**: Interactive project cards with detailed case studies
- **Timeline View**: Professional journey timeline with detailed experience sections
- **Skills Showcase**: Categorized technical expertise and tools
- **Community Section**: Events, contributions, and network
- **Testimonials**: Client and colleague reviews with carousel
- **Contact Form**: Functional contact form with email integration
- **SEO Optimized**: Proper meta tags and structured data

## Running the Project

### Prerequisites
- Node.js (v18 or higher)
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/jackie-portfolio.git
cd jackie-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

### Commands

- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run lint`: Run code linting
- `npm run preview`: Preview production build

## Deployment

This portfolio is ready for deployment on platforms like:

- **Vercel**: Recommended for React/Next.js applications
- **Netlify**: Excellent for static sites with serverless functions
- **AWS S3 + CloudFront**: Scalable and cost-effective option
- **GitHub Pages**: Simple deployment from GitHub repository

### Vercel Deployment (Recommended)

```bash
# Install Vercel CLI
npm i -g vercel

# Login to Vercel
vercel login

# Deploy
vercel --prod
```
